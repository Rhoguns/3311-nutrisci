NutriSci Nutrition Tracking Application

Overview

NutriSci is a comprehensive nutrition tracking application that helps users monitor their daily food intake, exercise activities, and nutritional goals. The application provides detailed analysis of macronutrients, meal comparisons, food swapping recommendations, and compliance with the Canada Food Guide.

Main Features

  • Profile Management - Create and manage user profiles with personal information
  • Meal Logging - Track daily meals with detailed ingredient information
  • Exercise Tracking - Log physical activities and calorie expenditure
  • Nutritional Analysis - Analyze daily intake and generate reports
  • Meal Comparison - Compare nutritional content between different meals
  • Food Swapping - Get recommendations for healthier food alternatives
  • CFG Compliance - Monitor adherence to Canada Food Guide recommendations
  • Data Visualization - Generate charts and graphs for nutritional insights

Package Structure

Technology Stack

  • Java 11+ - Core programming language
  • Swing - GUI framework
  • MySQL - Database system
  • JFreeChart - Chart generation library
  • JUnit 5 - Testing framework
  • Mockito - Mocking framework for tests
  • Maven - Build and dependency management

Getting Started

To use the NutriSci application:

  1. Run the main application from com.nutrisci.app.MealMain or com.nutrisci.app.ProfileMain
  2. Create a user profile with your personal information
  3. Start logging meals and exercises
  4. Use the analysis features to track your nutritional progress

Test Coverage

The application includes comprehensive test coverage with Test-Driven Development (TDD) approach, covering test cases TC01-TC12 for all major functionality including profile management, meal logging, exercise tracking, and nutritional analysis.
